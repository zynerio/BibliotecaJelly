# Biblioteca Jelly v2.1

## Resumen
Biblioteca Jelly 2.1 consolida en una unica release publica la evolucion completa de la rama 2.x. Se mantiene el modo avanzado por bibliotecas y se completa el modo clasico con soporte robusto para contenido "Otros" (video/foto), junto a mejoras de sincronizacion y consistencia de datos.

## Novedades principales
- Pestaña "Otros" en modo clasico:
  - agrupacion ligera por tipo (Videos / Imagenes),
  - filtro por biblioteca dentro de la pestaña,
  - modal con busqueda local y orden (A-Z / Ultimos añadidos).
- Sincronizacion de "Otros" por alcance:
  - Normal,
  - Rapida,
  - Ultimos añadidos,
  - inclusion en "Todo".
- Correccion de "Marcar vistas" para limpiar novedades de forma fiable incluso con desfases de hora servidor/dispositivo.
- Reconciliacion de eliminados en other_media durante sincronizacion completa para mantener el catalogo local consistente.

## Cambios tecnicos destacados
- Version de app actualizada a 2.1 (versionCode 8).
- Ajustes de repositorio para sync de Others por scope.
- Mejoras de UX en modo clasico y avanzadas de bibliotecas.
- Internacionalizacion ES/EN actualizada para nuevos flujos.

## Compatibilidad
- Android minimo: 7.0 (API 24)
- Stack: Kotlin + Jetpack Compose + Material 3

## Nota de consolidacion
La v2.0 se uso como base interna y no se publica por separado; sus mejoras quedan incluidas en esta v2.1.
